Inherits from UTCSTableViewController : UTCSContentViewController : UIViewController
Declared in UTCSHeaderTableViewController.h

Overview

UTCSHeaderTableViewController is an abstract view controller that extends the functionality of UTCSTableViewController by managing its own header. UTCSHeaderTableViewController also provides an additional image view intended to add a seemingly-dynamic blurring effect.
