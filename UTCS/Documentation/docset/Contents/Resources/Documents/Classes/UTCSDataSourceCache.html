UTCSDataSourceCache Class Reference

Inherits from NSObject
Declared in UTCSDataSourceCache.h

Overview

UTCSDataSourceCache is an abstract class that provides a simple-to-use API for a data source to cache objects.

While UTCSDataSourceCache does support the caching of single objects, it is intended to be used to cache collections, such as NSArray or NSDictionary.
